Pasos Para crear tu servidor.
1. Ejecuta MCST-setup.sh (./MCST-setup.sh)
2. Cuando finalize, ejecuta MCST.sh o MCST-H.sh (./MCST.sh)
3. Abre el servidor y cierralo (opcion [1] y escribe stop para pararlo)
4. Si quieres plugins, descargalos de spigot o una fuente
oficial y ponlos en la carpeta "plugins".
5. Abre de nuevo el servidor y ya está listo. (opcion [1])

Cualquier configuración de plugins o del servidor se puede
modificar en server.properties y en las carpetas de los plugins
y sus archivos correspondientes.

----ENGLISH----

Steps to create your server.
1. Run MCST-setup.sh (./MCST-setup.sh)
2. When finished, run MCST.sh or MCST-H.sh (./MCST.sh)
3. Open the server and close it (option [1] and type stop to stop it)
4. If you want plugins, download them from spigot or a font
and put them in the "plugins" folder.
5. Open the server again and you're good to go. (option [1])

Any plugin or server configuration can be
modify in server.properties and plugin folders
and their corresponding files.
